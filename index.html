About

The goal of this project is to create a machine learning model that can accurately predict the likelihood and intensity of tornadic activity, based on the real-time data collected during storm chases, including but not limited to, chases I physically am present for and participate in.

Usage

To use the Dorothy AI system, simply download the code from our GitHub repository and follow the instructions provided in the README file. We recommend using Google Colab for running the machine learning algorithms and Jupyter Notebook for data analysis.

Contributing

Contributions are welcome from anyone interested in improving the Dorothy AI project. If you have experience in machine learning, data analysis, or meteorology, feel free to contribute by submitting pull requests or opening issues on our GitHub repository.
